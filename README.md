# AI Resume Analyzer

## Description

AI Resume Analyzer is a web application that analyzes resumes and provides insights and recommendations. It includes secure user authentication and a personalized dashboard.

## Features

* User Registration
* Secure Login
* Logout Functionality
* Resume Upload
* Resume Analysis
* Dashboard
* Recent Reports
* Password Hashing using bcrypt
* SQLite Database
* Cloud Deployment using Render

## Technologies Used

* Python
* Flask
* HTML
* CSS
* Bootstrap
* SQLite
* Jinja2
* bcrypt
